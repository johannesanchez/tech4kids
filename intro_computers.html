What is a Computer?

A computer is a device that can perform a variety of tasks by following instructions (programs). Computers are used to help solve problems, create art, play games, and communicate with others. What is Scratch?

Scratch is a programming language designed for kids to create animations, games, and interactive stories. Scratch makes it easy and fun to learn the basics of coding through its visual interface, where you can drag and drop code blocks to create programs.
